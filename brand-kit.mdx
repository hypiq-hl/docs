## Typography

HYPIQ uses modern, clean typography that ensures excellent readability across all platforms.

### Primary Font
**Inter** - Used for all body text, headings, and interface elements
- Clean, modern sans-serif
- Excellent readability
- Professional appearance

### Font Weights
- **Light (300)** - For subtle text
- **Regular (400)** - For body text
- **Medium (500)** - For emphasis
- **Semi-bold (600)** - For headings
- **Bold (700)** - For strong emphasis

## Brand Guidelines

<AccordionGroup>
  <Accordion title="Logo Usage">
    **Do:**
    - Use the official logo files provided
    - Maintain proper clearspace around the logo
    - Use on appropriate contrast backgrounds
    - Scale proportionally
    
    **Don't:**
    - Modify logo colors or proportions
    - Add effects or distortions
    - Use low-resolution versions
    - Place on busy backgrounds
  </Accordion>

  <Accordion title="Color Guidelines">
    **Primary Colors:**
    Use our primary green (#4ade80) for:
    - Call-to-action buttons
    - Interactive elements
    - Accent highlights
    - Brand emphasis
    
    **Supporting Colors:**
    - Dark backgrounds for professional appearance
    - Light text for readability
    - Subtle borders and dividers
  </Accordion>

  <Accordion title="Voice & Tone">
    **HYPIQ Brand Voice:**
    - Professional yet approachable
    - Innovative and forward-thinking
    - Clear and informative
    - Trustworthy and reliable
    
    **Tone Guidelines:**
    - Use clear, concise language
    - Avoid jargon when possible
    - Be helpful and supportive
    - Maintain professional standards
  </Accordion>
</AccordionGroup>
