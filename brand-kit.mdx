### Logo Usage Guidelines
<Accordion title="Proper Usage">
  **Do:**
  - Use the official logo files provided
  - Maintain proper clearspace around the logo (minimum 2x logo height)
  - Use on appropriate contrast backgrounds
  - Scale proportionally to maintain aspect ratio
  
  **Don't:**
  - Modify logo colors or proportions
  - Add effects, shadows, or distortions
  - Use low-resolution versions for print
  - Place on busy or low-contrast backgrounds
</Accordion>

### Icon Usage Guidelines
<Accordion title="Best Practices">
  **Sizing:**
  - 16px for small UI elements
  - 20px for navigation items
  - 24px for standard interface icons
  - 32px+ for featured content icons
  
  **Color:**
  - Use HYPIQ Green (#4ade80) for active/primary states
  - Use Text Secondary (#b0b8bf) for inactive states
  - Use Text Primary (#ffffff) for navigation icons
  
  **Spacing:**
  - Maintain consistent padding around icons
  - Align icons to text baseline when inline
  - Use consistent icon-to-text spacing (8px recommended)
</Accordion>
