### Typography Scale

<Accordion title="Size & Weight Specifications">
  **Heading Hierarchy:**
  ```
  H1: 32px (2rem) - Extra Bold (800) - Leading: 1.2
  H2: 24px (1.5rem) - Bold (700) - Leading: 1.3
  H3: 20px (1.25rem) - Semi-bold (600) - Leading: 1.4
  H4: 18px (1.125rem) - Semi-bold (600) - Leading: 1.4
  ```
  
  **Body Text:**
  ```
  Large: 18px (1.125rem) - Regular (400) - Leading: 1.6
  Base: 16px (1rem) - Regular (400) - Leading: 1.6
  Small: 14px (0.875rem) - Regular (400) - Leading: 1.5
  ```
  
  **Supporting Text:**
  ```
  Caption: 12px (0.75rem) - Light (300) - Leading: 1.4
  Overline: 11px (0.6875rem) - Medium (500) - Leading: 1.3
  ```
</Accordion>

### Typography Best Practices

<Accordion title="Implementation Guidelines">
  **Line Height (Leading):**
  - Body text: 1.6 for optimal readability
  - Headings: 1.2-1.4 for tight, impactful spacing
  - Captions: 1.4 for compact information display
  
  **Letter Spacing:**
  - Default: 0 (Inter's built-in spacing)
  - Overlines/Labels: +0.05em for improved legibility
  - Never use negative letter-spacing
  
  **Paragraph Spacing:**
  - 16px between paragraphs
  - 24px before section headings
  - 8px after headings before content
  
  **Line Length:**
  - Optimal: 45-75 characters per line
  - Maximum: 85 characters
  - Mobile: 35-45 characters
</Accordion>

### Icon Sizing Guidelines

<Accordion title="Size Specifications">
  **Standard Sizes:**
  ```
  Small: 16px - Inline with text, compact UI
  Default: 20px - Navigation, form elements
  Medium: 24px - Standard interface icons
  Large: 32px - Featured content, headers
  XL: 48px+ - Hero sections, illustrations
  ```
  
  **Usage Context:**
  - **16px**: Menu items, inline text icons, dense layouts
  - **20px**: Navigation bars, sidebar items, form inputs
  - **24px**: Buttons, cards, standard interface elements
  - **32px**: Page headers, section titles, featured content
  - **48px+**: Landing pages, empty states, illustrations
</Accordion>


  ```css
  .icon {
  ```
