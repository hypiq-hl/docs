## Usage Guidelines

### Logo Clearspace
<Accordion title="Minimum Spacing Requirements">
  **Clearspace Rule**: Maintain minimum clearspace equal to 2x the logo height around all sides of the logo
  
  **Why It Matters**: 
  - Ensures logo visibility and impact
  - Prevents visual clutter
  - Maintains brand integrity
  - Improves legibility across all applications
  
  **Application**: Apply clearspace to all logo placements including websites, documents, merchandise, and advertising materials
</Accordion>

### Proper Usage
<Accordion title="Do's and Don'ts">
  **‚úÖ DO:**
  - Use official logo files provided

  - Use on appropriate contrast backgrounds
  - Scale proportionally to maintain aspect ratio
  - Use the white version on dark backgrounds

</Accordion>

## File Formats & Sizes

### Digital Applications
```
PNG Files:
- 512x512px (High resolution)
- 256x256px (Standard web)
- 128x128px (Small applications)
- 64x64px (Icons)
- 32x32px (Favicons)

SVG Files:
- Scalable vector format
- Perfect for responsive design
- Retina display ready
- Print applications
```

### Print Applications
```
Vector Formats:
- SVG (web-friendly vector)
- AI (Adobe Illustrator)
- EPS (print production)
- PDF (document embedding)

Resolution Requirements:
- Minimum 300 DPI for print
- Vector preferred for scalability
- CMYK color mode for offset printing
```

## Co-branding Guidelines

<Accordion title="Partner Logo Usage">
  **When using HYPIQ logo with partner brands:**
  
  **Hierarchy**: HYPIQ logo should maintain equal or greater prominence
  
  **Spacing**: Maintain adequate separation between logos (minimum 3x logo height)
  
  **Alignment**: Align logos consistently (baseline, center, or top alignment)
  
  **Approval**: All co-branding applications require prior approval from HYPIQ brand team
  
  **Contact**: [brand@hypiq.finance](mailto:brand@hypiq.finance) for co-branding requests
