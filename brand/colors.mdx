## Color Usage Guidelines

### Primary Color Applications
<Accordion title="HYPIQ Green (#4ade80)">
  **Primary Usage:**
  - Call-to-action buttons
  - Interactive elements (active states)
  - Brand accents and highlights
  - Progress indicators
  - Success confirmations
  
  **Best Practices:**
  - Use sparingly to maintain impact
  - Ensure sufficient contrast with backgrounds
  - Pair with neutral colors for balance
  - Test accessibility compliance (WCAG AA)
</Accordion>

### Background Color System
<Accordion title="Dark Theme Hierarchy">
  **Layer Structure:**
  1. **Dark Primary (#0a1a15)**: Main page background
  2. **Dark Secondary (#0f1f1a)**: Cards, panels, modals
  3. **Dark Tertiary (#162b20)**: Borders, dividers, subtle elements
  
  **Implementation:**
  - Use layered approach for depth perception
  - Maintain consistent elevation system
  - Apply subtle transitions between layers
</Accordion>

### Text Color Hierarchy
<Accordion title="Reading Hierarchy">
  **Content Priority:**
  1. **Primary (#ffffff)**: Main headings, body text, important content
  2. **Secondary (#b0b8bf)**: Subheadings, captions, less critical information
  3. **Muted (#7a8085)**: Placeholder text, disabled states, metadata
  
  **Accessibility:**
  - All combinations meet WCAG AA contrast requirements
  - Test with colorblind simulators
  - Consider focus states and hover effects
</Accordion>

## Implementation Guidelines

### CSS Variables
```css
:root {
  /* Primary Colors */
  --color-primary: #4ade80;
  --color-primary-dark: #22c55e;
  --color-success: #16a34a;
  
  /* Background Colors */
  --bg-primary: #0a1a15;
  --bg-secondary: #0f1f1a;
  --bg-tertiary: #162b20;
  
  /* Text Colors */
  --text-primary: #ffffff;
  --text-secondary: #b0b8bf;
  --text-muted: #7a8085;
  
  /* Status Colors */
  --color-warning: #f59e0b;
  --color-error: #ef4444;
  --color-info: #3b82f6;
  --color-neutral: #6b7280;
}
```

### Design System Integration
<Accordion title="Component Color Mapping">
  **Buttons:**
  - Primary: HYPIQ Green background, Dark Primary text
  - Secondary: Dark Secondary background, Text Primary
  - Destructive: Error background, White text
  
  **Form Elements:**
  - Inputs: Dark Secondary background, Text Primary
  - Focus: HYPIQ Green border, maintaining accessibility
  - Error: Error border and text color
  
  **Navigation:**
  - Active states: HYPIQ Green accents
  - Hover states: Dark Tertiary backgrounds
  - Text: Primary and Secondary text colors
</Accordion>

<Info>
  Always test color combinations for accessibility compliance. Use tools like WebAIM's Contrast Checker to verify WCAG guidelines are met.
</Info>

<Warning>
  Never use color as the only way to convey information. Always provide additional visual or textual cues for important states or actions.
</Warning>
